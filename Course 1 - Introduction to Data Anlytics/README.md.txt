Course 1 - Introduction to Data Analytics:

In this course, Data Analysis, the role of a Data Analyst, and the tools used in this job are introduced. It describes about the skills and responsibilities of a data analyst. This course differentiates between the roles of Data Analysts, Data Scientists, and Data Engineers. It familiarized with the data ecosystem, alongside Databases, Data Warehouses, Data Marts, Data Lakes and Data Pipelines.

As the project, a Credit Card Fraud Transactional Data analysis was done to understand the fundamentals of the data analysis process including gathering, cleaning, analyzing and sharing data and communicating the insights with the use of visualizations and dashboard tools in Excel.
