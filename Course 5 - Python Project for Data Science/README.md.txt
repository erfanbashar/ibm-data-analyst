Course 5 - Python Project for Data Science:

About this Course:
This course is intended to demonstrate foundational Python skills for working with data. It primarily involves completing a project associated with the role of a Data Scientist or a Data Analyst and be provided with a real-world data set and a real-world inspired scenario to identify patterns and trends. 

It includes specific data science and data analytics tasks such as extracting data, web scraping, visualizing data and creating a dashboard. It requires proficiency with Python and using libraries such as Pandas and Beautiful Soup within a Jupyter Notebook.

Module 1:
In this module, the following skill is applied - Python fundamentals, Python data structures, and work with data in Python.

Objectives:
* Perform webscraping in Python to obtain data.
* Extract data by using a Python library.
